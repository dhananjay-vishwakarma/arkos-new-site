# ARKOS Header HTML/CSS

This package contains two implementations:

- `index.html` + `styles.css`: pixel-matched version using the supplied reference artwork as a background asset. Use this when you need the exact same visual.
- `editable.html` + `editable.css`: editable HTML/CSS/SVG rebuild with real nav text and icons. It uses the generated `assets/arkos-center-panel.png` logo/center badge asset because the original shield/logo is artwork.

Assets generated from the supplied image:

- `assets/header-option-3-full.png`
- `assets/header-option-3-header-only.png`
- `assets/arkos-center-panel.png`
- `assets/arkos-logo-lockup-crop.png`
